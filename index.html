Supported Formats

Markdown (.md)

Supports standard Markdown syntax:

  • Headers: # through ######
  • Emphasis: **bold**, *italic*
  • Links: [text](url)
  • Images: ![alt](url)
  • Code Blocks: Triple backticks (```)
  • Inline Code: Single backticks (`)
  • Blockquotes: >
  • Horizontal Rules: ---
  • Paragraphs: Double line breaks.
  • Line Breaks: Single line breaks within a paragraph convert to <br>.

HTML (.html)

Supports standard HTML tags:

  • <h1>, <p>, <div>, <span>
  • <b>Bold</b>, <i>Italic</i>
  • <br> for line breaks.
  • <img src="url"> (External)
Note: <script> tags are removed for security.

CSV (.csv)

Comma Separated Values.

  • Previewed as an interactive HTML table.
  • First line is treated as Header row.
  • Supports basic cell rendering.

TXT / Hybrid Text (.txt)

Standard text files with media embedding support.

  • Preview uses Monospace font.
  • Media: Use the "Add Media" button in the editor toolbar.
  • Supported Media: Images, Audio, and Video (MP4, etc.).
  • Media files are converted to Base64 and embedded directly in the file.
